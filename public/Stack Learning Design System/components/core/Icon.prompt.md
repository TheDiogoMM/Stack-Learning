Stack Learning line icon — minimalist 24×24 stroke icon that inherits the current text color; use anywhere the product needs a glyph (replaces all emoji).

```jsx
<Icon name="check" size={18} style={{ color: 'var(--color-success)' }} />
<Icon name="play" />
<Icon name="bookmark-fill" style={{ color: 'var(--accent-secondary)' }} />
```

Names: check, check-circle, lock, play, bookmark, bookmark-fill, bulb, mail, phone, clock, chevron-left, chevron-right, arrow-right, arrow-left, code, terminal, book, layers, search, user, menu, external, sparkle, plus, x, circle-dot. Props: `name`, `size` (px, default 20), `strokeWidth` (default 1.6). Color comes from CSS `color`.
