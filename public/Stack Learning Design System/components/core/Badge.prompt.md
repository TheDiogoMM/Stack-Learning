Compact category / status label using the brand "tone" pattern — a colored border over a 13% tint of the same color. Use for technology, difficulty, and metadata chips.

```jsx
<Badge tone="terracota">React 19</Badge>
<Badge tone="beginner">Iniciante</Badge>
<Badge tone="neutral">45 min</Badge>
<Badge tone="supabase" solid>Novo</Badge>
```

Tones: brand (`terracota`, `gold`, `neutral`), difficulty (`beginner`, `intermediate`, `advanced`), technology (`react`, `nextjs`, `typescript`, `tailwind`, `shadcn`, `supabase`). `color` overrides any tone; `solid` fills instead of tinting.
