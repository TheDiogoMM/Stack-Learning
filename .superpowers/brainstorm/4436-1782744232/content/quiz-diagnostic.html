Embeddings e busca semântica
3 perguntas
Qual operação é usada para medir a similaridade entre dois vetores de embedding?
A
Produto escalar normalizado (cosine similarity)
B
Distância euclidiana entre os vetores
C
Soma de todos os elementos do vetor
✓ A resposta é A — cosine similarity
A similaridade do cosseno mede o ângulo entre vetores, sendo insensível à magnitude — ideal para comparar embeddings semânticos.
